Quick Start
1

Add Your Fireworks.ai API Keys

Go to the API Keys page and add one or more Fireworks.ai API keys. The service will automatically rotate between them.

2

Use OpenAI SDK

Simply change the base URL in your OpenAI client:

from openai import OpenAI client = OpenAI( base_url="https://api-provider-service-production-6e52.up.railway.app/v1", api_key="not-needed" ) response = client.chat.completions.create( model="accounts/fireworks/models/llama-v3p1-8b-instruct", messages=[{"role": "user", "content": "Hello!"}] ) print(response.choices[0].message.content)
3

Or Use Anthropic SDK

Works with Anthropic SDK too:

import anthropic client = anthropic.Anthropic( base_url="https://api-provider-service-production-6e52.up.railway.app", api_key="not-needed" ) response = client.messages.create( model="accounts/fireworks/models/kimi-k2p5", messages=[{"role": "user", "content": "Hello!"}], max_tokens=256 ) print(response.content[0].text)
4

Or Use cURL

Direct HTTP requests work too:

curl https://api-provider-service-production-6e52.up.railway.app/v1/chat/completions \ -H "Content-Type: application/json" \ -d '{ "model": "accounts/fireworks/models/llama-v3p1-8b-instruct", "messages": [{"role": "user", "content": "Hello!"}] }'
API Endpoints
POST /v1/chat/completions OpenAI chat completions
POST /v1/completions OpenAI completions
POST /v1/messages Anthropic messages
GET /health Health check
Configuration

The service can be configured via:

  • config/config.yaml - Main configuration file
  • .env - Environment variables
  • Dashboard Settings page - Web interface

Default Settings

  • Port: 10736 (configurable)
  • Dashboard Auth: admin / macro
  • Key Rotation: Enabled
  • Auto Backup: Every 10 hours
  • Rate Limiting: Disabled (optional)

Built with Flask, SQLite, and Fireworks.ai
